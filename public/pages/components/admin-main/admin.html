Користувачі

Видалення доступне лише для користувачів без пов’язаних дописів, відгуків і звітів.

Дописи

Тут відображаються всі запити на допомогу та збори коштів, доступні для модерації.

Відгуки

Повний список відгуків допомагає бачити зв’язок між волонтерами та конкретними запитами.

Звіти

Звіти показують завершені етапи взаємодії та дозволяють переглянути підтвердження допомоги.
