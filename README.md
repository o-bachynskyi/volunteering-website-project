# Volunteering website project

## Quick start

1. Copy `.env.example` to `.env` and fill in your PostgreSQL credentials.
2. Run `npm install`.
3. Run `npm run db:init` to create the database schema.
4. Run `npm start`.
5. Open `http://localhost:3000`.

## Useful scripts

- `npm start` - start the server
- `npm run db:init` - create the database and schema
- `npm run demo` - run an automated demo scenario for auth, profile, posts, responses, and reports
